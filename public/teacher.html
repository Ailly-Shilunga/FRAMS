How This Works

Teacher Login

Teachers log in through the login page with their credentials. Once authenticated, the system retrieves their profile information and assigned classes from the database.

Student Data

Student information is stored in your database when they're added via the "Add Student" page. Each student gets a unique ID (like SID-202301) that's generated automatically.

Attendance Tracking

The system uses facial recognition to automatically mark attendance when students enter the classroom. Teachers can manually override statuses as needed in this register.

Data Flow

1. Teacher logs in → 2. System fetches teacher's classes → 3. Teacher selects class/date → 4. System loads student list with attendance data → 5. Teacher can modify and save attendance records.
